# traceskeleton.go

Golang version using prallel goroutines.

## Usage

```
import "traceskeleton"
```

See `example.go` for detailed usage.

All source code is in `src/traceskeleton/traceskeleton.go`

## Compile

```
export GO_PATH=$PWD; go build example.go
```
